## Project Overview

Smart Cafeteria Management System is a web-based application developed to streamline cafeteria operations. The system enables customers to browse menus, place orders, manage carts, and submit reviews, while staff and administrators can manage orders, inventory, queues, and users.

## Features

### Customer

* User registration and login
* Browse food and beverage menu
* Add items to cart
* Place orders
* View order history
* Submit reviews

### Staff

* Manage orders
* Queue management
* Staff dashboard

### Administrator

* Manage menu items
* Inventory management
* User management
* Order management
* Review management
* Administrative dashboard
